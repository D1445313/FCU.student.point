# 逢甲大學學業進度與時間管理整合系統 - 產品需求文件 (PRD)

## 1. 專案概述

### 1.1 背景與動機
在大學求學階段，學生需要面對多樣化的課業壓力與時間分配。然而，許多學生缺乏整合性工具來管理課業、學分與每日行程，常因規劃不當影響學習效率與畢業進度。市面上的工具（如行事曆、待辦清單、記帳軟體）多半功能單一，且未整合逢甲大學特有的學分規則、GPA 換算與畢業門檻審查。
因此，本專題專為**逢甲大學學生**設計一套整合系統，結合成績紀錄、學分加總、GPA 換算、時間行程規劃與學分預警系統，協助學生清楚掌握學習進度，並有效安排每日行程，提升學習效率。

### 1.2 目標用戶
* **在校大學生**：需要隨時追蹤已修學分與 GPA。
* **應屆畢業生**：需要進行最後的畢業資格自我審查，確保能順利畢業。
* **轉學生與復學生**：需要重新整理抵免學分、已修學分與後續修課計畫。

### 1.3 核心價值主張
提供一站式、隱私安全（支援本地端與伺服器端雙重防護）、體驗極佳的逢甲大學學業與時間管理助手，告別繁瑣的手動學分計算，並以圖像化介面提供直覺的進度回饋。

## 2. 功能需求 (Functional Requirements)

系統主要分為四個核心模組，本次開發將由我們小組成員分工合作，其中重點實作 **F-01：學分分類錄入**。

| 功能編號 | 功能名稱 | 功能描述 | 負責人 |
| :--- | :--- | :--- | :--- |
| **F-01** | **學分分類錄入** | 支援手動輸入或批次匯入（CSV 範本與 MyFCU 歷年成績文字貼上解析）課程名稱、學分、成績及類別（必修、選修、通識領域）。 | 郭芮妤、高晨軒 |
| **F-02** | **畢業門檻設定** | 使用者可自定義各類別所需總學分（如：通識 20 學分、系必修 60 學分、總畢業學分 128 學分）。 | 王郁棻、林庭玉 |
| **F-03** | **自動化試算引擎** | 自動計算已完成、修習中、待修習學分；支援百分制與等級制 GPA（4.0/4.3）轉換計算。 | 黃郁婷、賴佩妤 |
| **F-04** | **學分預警與視覺化** | 透過圖表將學分數據轉化為直覺的進度資訊，並針對潛在的畢業危機自動發出預警。 | 羅方妤 |

### 2.1 使用者故事 (User Stories)
* **手動錄入**：作為一名學生，我希望能夠手動新增一筆課程資訊（包含學 semester、課程名稱、學分數、分數/成績、類別），以便記錄我這學期修習的單門課程。
* **批次匯入 (CSV)**：作為一名已經修了許多課程的三年級學生，我希望能夠透過上傳 CSV 檔案一次匯入所有歷史課程，以便免去一筆筆手動輸入的繁瑣程序。
* **貼上解析 (FCU Parser)**：作為一名懶得整理 CSV 的轉學生，我希望可以直接從 MyFCU 歷年成績網頁中複製表格文字，並貼入系統，讓系統自動辨識並列出所有課程，以便一鍵確認匯入。
* **課程管理 (CRUD)**：作為一名錄入錯誤的使用者，我希望能夠隨時修改或刪除已錄入的課程，以確保學分統計數據的正確性。

## 3. 非功能需求 (Non-functional Requirements)

### 3.1 技術限制
* **前端**：HTML / CSS / JavaScript（純前端，不使用 Vue/React 框架，遵循課堂統一規範）。
* **後端**：Python + Flask 框架。
* **資料庫**：SQLite（本機輕量資料庫 `instance/database.db`），搭配前端 LocalStorage 進行備份，保護隱私不強制上傳至公開伺服器。

### 3.2 效能與品質
* **回應速度**：學分加總與 GPA 計算需在 1 秒內完成，並即時更新介面。
* **載入時間**：首頁在行動裝置與電腦之載入時間控制在 2 秒內。
* **容錯性**：使用者輸入負數或非數字學分時，表單需進行驗證並提供明確的錯誤提示。
* **響應式排版 (RWD)**：頁面必須完美適應手機、平板與桌機。

## 4. MVP 範圍 (Minimum Viable Product)

* **Must Have (必須有)**：
  - 手動課程新增表單、欄位格式驗證。
  - 已錄入課程列表，提供搜尋與篩選。
  - 修改與刪除課程功能。
  - 標準 CSV 格式之匯入與匯出。
* **Should Have (應該有)**：
  - MyFCU 歷年成績文字貼上解析器（Regex-based 前端/後端解析）。
  - 本地 LocalStorage 備份與還原功能。
* **Nice to Have (以後再說)**：
  - AI 排課建議與學期課表排程。
  - 多人帳號線上雲端同步資料庫。
