# 逢甲大學畢業門檻與時間管理整合系統 (FCU Graduation Requirements Setting)

本系統是專為逢甲大學 (FCU) 學生設計的畢業門檻自訂與學分修課時間管理系統。使用者可以依據自身就讀學系之規定，自訂畢業學分門檻分類與目標，並透過視覺化儀表板追蹤修課學分數、非學分指標（英文檢定、服務學習、學術倫理、體育），以及依學期規劃未來的修課藍圖。

---

## 🌟 系統特色

1. **自訂畢業學分分類 (Graduation Threshold Setting)**
   - 使用者可自由新增、修改、刪除不同的學分分類（如：通識、系必修、專業選修、自由選修等），並為每個分類設定目標學分數。
   - 設有智慧防呆校驗：當所有自訂分類的目標學分加總與設定的「畢業總學分目標（如 128 學分）」不符時，系統會自動在桌面上顯示警告提示。
2. **快速載入逢甲學系範本**
   - 內建逢甲大學通用畢業學分範本、資訊工程學系 (IECS) 畢業學分範本以及商學院 (CO) 通用範本，一鍵預載標準修課要求。
3. **修課與學期規劃藍圖 (Semester Planner & Warning)**
   - 依學期（110-1 至 115-2）分組管理修課歷程，動態計算單學期修習學分數。
   - 支援逢甲學分上限/下限預警功能（單學期超過 25 學分顯示「超修警告」，低於 12 學分顯示「低限警示」），幫助學生符合選課規範。
4. **非學分畢業門檻檢核**
   - 整合逢甲學生必備的非學分畢業條件：英文畢業門檻、服務學習（一、二）、學術倫理教育課程與體育大一大二門檻。
5. **資料存取與備份功能**
   - 自動將資料儲存於瀏覽器的 `localStorage` 中，關閉網頁資料不遺失。
   - 提供 JSON 檔案匯出備份與匯入還原功能，方便將規劃同步至其他裝置。
6. **極致視覺美學設計**
   - 採用毛玻璃質感（Glassmorphism）與深色模式配色的漸層光暈背景。
   - 支援一鍵切換深色/淺色主題。
   - 使用 SVG 動態圓環進度條繪製修課達成率。

## 🛠️ 技術疊代與架構

- **結構與標記**: HTML5 語意化標籤，結構清晰，符合 SEO 與無障礙瀏覽規範。
- **樣式與設計**: CSS3 (Vanilla CSS)，採用 CSS 變數進行主題切換，全響應式設計 (RWD)，相容行動版與桌面端。
- **邏輯與控制**: 原生 JavaScript (ES6)，無額外框架依賴，加載飛快。
- **資料儲存**: 本地 `localStorage`，保證隱私與離線可用。

## 🚀 啟動與使用說明

### 1. 本地啟動
本專案為純前端單頁應用程式 (SPA)，不需複雜安裝即可運行：
- 直接在瀏覽器中雙擊打開 [index.html](file:///c:/Users/Elin%20Wang/Downloads/agents/.agents/skills/FCU.student.point/index.html)。
- 或者，您可以使用 VS Code 的 `Live Server` 插件，或在該資料夾下透過 Terminal 執行簡易 Server 啟動（如 `npx serve .` 或 `python -m http.server`）。

### 2. 操作說明
- **套用範本**: 在網頁右上方點擊下拉選單「快速套用學校門檻範本...」，即可加載範例設定。
- **修改總學分**: 點擊畢業總進度卡片右側的「編輯目標」即可調整。
- **新增課程**: 在左側的「新增修課紀錄」中填入名稱、學分與分類，儲存後右側學期卡片將自動更新。
- **修改或刪除**: 將滑鼠移至學期中的課程小卡上，會浮現「編輯」與「刪除」圖示以進行操作；點擊學分分類表格中的「編輯」也可直接微調分類目標與背景顏色。
